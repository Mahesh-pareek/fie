# FIE — Financial Intelligence Engine (v1)

A local-first CLI tool to turn bank statements into
reviewable, trustworthy personal financial intelligence.

## Features
- PDF ingestion (Canara Bank)
- Deterministic micro-transaction rules
- Human-in-the-loop review
- Editable transaction classification
- Scope-wise and category-wise summaries
- Queryable transaction ledger

## Philosophy
FIE does not guess.  
Only reviewed personal transactions are counted.

## Status
v1 — Core engine complete.
